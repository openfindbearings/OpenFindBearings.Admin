# 同步管理模块 v1.0.0

## 概述

同步管理模块提供对 FindBearings.Sync ETL 管线的远程管理，包括状态监控、手动触发各阶段、重试失败记录、查看日志和统计概览。

## ETL 状态机

```
Idle
  │
  ├─→ Extracting ──→ Idle（E 完成）
  ├─→ Transforming──→ Idle（T 完成）
  ├─→ Loading ──────→ Idle（L 完成）
  ├─→ Retrying ─────→ Idle（重试完成）
  └─→ StuckCheck ───→ Idle（卡住检查完成）
```

## API 调用序列

```
1. 查看 ETL 状态
   GET /api/etl/status
   ← {currentPhase: "Idle", lastRunAt: "...", ...}

2. 触发全量 ETL
   POST /api/etl/run
   ← {message: "全量 ETL 已触发"}

   同步触发后前端轮询 status 页面，通过 SignalR 推送进度

3. 重试失败记录
   POST /api/etl/retry
   ← {message: "重试已触发"}
```

## 降级策略

| 场景 | 表现 |
|------|------|
| Sync API 离线 | 页面显示"Sync 服务不可用"，触发按钮全部禁用 |
| ETL 阶段执行中 | 按钮禁用，显示当前阶段和进度 |
| 触发后无响应 | 30 秒超时提示"触发失败，请稍后重试" |
