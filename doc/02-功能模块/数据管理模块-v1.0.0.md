# 数据管理模块 v1.0.0

## 概述

数据管理模块提供对 OpenFindBearings.Api 中所有业务数据的浏览、搜索、编辑和删除功能。包括品牌、轴承类型、轴承、商家和替代品五个子模块。

## 1. 品牌管理（/Data/Brands）

### 字段

| 字段 | 类型 | 可编辑 | 说明 |
|------|------|--------|------|
| Id | Guid | 否 | 主键 |
| Code | string(50) | 是 | 品牌代码（如 SKF, NSK, TMB） |
| ChineseName | string(200) | 是 | 中文名（如 斯凯孚） |
| EnglishName | string(200) | 是 | 英文名（如 SKF） |
| Description | text | 是 | 品牌描述 |
| LogoUrl | string(500) | 是 | Logo 图片 URL |
| Website | string(500) | 是 | 官网 |
| DataSource | object | 否 | 来源信息（只读） |
| CreatedAt | datetime | 否 | 创建时间 |
| UpdatedAt | datetime | 否 | 最后更新时间 |

### 功能

- 品牌列表：分页，支持 Code/ChineseName/EnglishName 搜索
- 新增品牌：表单填写 Code + ChineseName + EnglishName（Code 必填且唯一）
- 编辑品牌：修改名称、描述、Logo、官网
- 删除品牌：软删除（级联检查是否有轴承引用该品牌）

### 操作限制

- Code 创建后不可修改（影响其他实体的引用）
- 有轴承引用的品牌不可删除（提示先移除关联轴承）

## 3. 轴承管理（/Data/Bearings）

### 字段

| 字段 | 类型 | 可编辑 | 说明 |
|------|------|--------|------|
| Id | Guid | 否 | 主键 |
| PartNumber | string(100) | 否 | 型号（不可修改，数据源权威） |
| OldNumber | string(100) | 是 | 曾用代号 |
| TypeName | string(100) | 是 | 类型名称 |
| BrandName | string(100) | 是 | 品牌名称 |
| Dimensions | string(200) | 是 | 外形尺寸（d×D×B） |
| InnerDiameter | decimal? | 是 | 内径(mm) |
| OuterDiameter | decimal? | 是 | 外径(mm) |
| Width | decimal? | 是 | 宽度(mm) |
| DynamicLoad | decimal? | 是 | 额定动载荷(kN) |
| StaticLoad | decimal? | 是 | 额定静载荷(kN) |
| LimitingSpeed | decimal? | 是 | 极限转速(r/min) |
| GreaseSpeed | decimal? | 是 | 脂润滑转速(r/min) |
| Weight | decimal? | 是 | 重量(kg) |
| Price | string(200) | 是 | 参考价 |
| Description | text | 是 | 产品描述 |
| Image1Url | string(500) | 是 | 主图 URL |
| Image2Url | string(500) | 是 | 副图 URL |
| Image3DUrl | string(500) | 是 | 3D 模型 URL |
| Image2DUrl | string(500) | 是 | 2D CAD URL |
| DataSource | object | 否 | 来源信息（只读） |
| IsOnSale | bool | 是 | 是否在售 |
| CreatedAt | datetime | 否 | 创建时间 |
| UpdatedAt | datetime | 否 | 最后更新时间 |

### 功能

- 轴承列表：分页 + 多条件搜索（PartNumber/Brand/Type/尺寸范围）
- 轴承详情：完整信息展示，含图片预览
- 编辑轴承：修改技术参数、价格、描述、图片 URL
- 删除轴承：确认后删除（级联检查商家关联和替代品）

### 搜索/筛选

```
搜索: [PartNumber 关键词]
品牌: [选择 ▼]
类型: [选择 ▼]
内径: [从____] ~ [到____] mm
外径: [从____] ~ [到____] mm
价格: [从____] ~ [到____] 元
状态: [全部 ▼]  数据源: [全部 ▼]
创建日期: [____] ~ [____]

[查询] [重置]
```

### 批量操作

- 批量修改品牌（选中轴承 → 修改品牌）
- 批量修改类型（选中轴承 → 修改类型）
- 批量删除（选中轴承 → 删除）
- 导出选中到 CSV

## 5. 替代品管理（/Data/Interchanges）

### 功能

- 替代品列表：分页 + PartNumber 搜索
- 新增替代关系：选择源轴承 + 目标轴承 + 替代类型（双向/单向）
- 删除替代关系

### 字段

| 字段 | 说明 |
|------|------|
| SourceBearingId | 源轴承 ID |
| TargetBearingId | 目标轴承 ID |
| SourcePartNumber | 源轴承型号（只读） |
| TargetPartNumber | 目标轴承型号（只读） |
| InterchangeType | Direct/Reverse/Bidirectional |
| DataSource | 来源信息（只读） |
