# 爬虫管理模块 v1.0.0

## 概述

爬虫管理模块提供对 FindBearings.Crawler 项目中所有爬虫的远程管理能力，包括状态查看、手动触发执行、调度配置和运行历史查询。

## API 调用序列

```
页面操作 → Admin Controller → CrawlerApiClient → Crawler API

1. 查看爬虫列表
   GET /api/crawlers
   ← [{name, displayName, lastRunAt, ...}]

2. 查看单个爬虫详情
   GET /api/crawlers/cbia_crawler/status
   ← {totalBearingCount, pendingDetailCount, lastRunAt, ...}

3. 手动触发爬取
   POST /api/crawlers/cbia_crawler/run
   ← {jobId: "xxx", startedAt: "2026-06-08T22:00:00Z"}

4. 轮询运行状态
   GET /api/crawlers/cbia_crawler/status (定期刷新)
   ← {lastRunAt, lastRunDuration, lastRunSuccess, ...}
```

## 降级策略

| 场景 | 表现 |
|------|------|
| Crawler 服务离线 | 页面显示"Crawler 服务不可用"，所有操作按钮禁用 |
| 触发运行后 Crawler 崩溃 | 下次健康检查标记离线，提醒管理员检查 |
| 长时间无响应 | 前端 30 秒超时提示"操作超时，请稍后重试" |
